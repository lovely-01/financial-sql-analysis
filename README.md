SQL–Excel Financial Performance & Cost Analysis Dashboard

->Project Overview
This project demonstrates an end-to-end analytics workflow using a normalized SQL database and an interactive Excel dashboard.
The objective was to analyse company-wise financial performance, cost efficiency, and department-level expense drivers across multiple years.

-> Data Model
The analysis is based on four SQL tables:
- Companies
- Financials
- Departments
- Department_Expenses
The database was designed with proper primary and foreign key relationships to support accurate financial analysis.

-> Tools & Skills Used
- SQL (table design, joins, aggregations, window functions)
- Microsoft Excel
- Pivot Tables & Charts
- Dashboard Design
- Data Analysis & Interpretation

 -> Dashboard Features
- Profit trend analysis across companies
- Cost efficiency comparison
- Department-wise expense breakdown
- Interactive filtering using slicers

-> Repository Structure
- SQL/ → Database schema and analysis queries
- Excel/ → Final Excel dashboard
- Dashboard_Screenshots/ → Dashboard visuals

-> Key Insights
- Several companies experienced recurring loss-making periods due to expense growth outpacing revenue.
- Cost efficiency varied significantly across companies, indicating differences in cost control.
- Sales and HR emerged as major cost-driving departments in multiple cases.
